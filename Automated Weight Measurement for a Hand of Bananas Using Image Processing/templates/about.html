About This Project

This project automates the process of estimating the weight of a hand of bananas using image processing and deep learning.

It utilizes a Convolutional Neural Network (CNN) trained on labeled images of banana hands. The frontend is built using Flask and styled with Bootstrap.

Designed to support agriculture and reduce manual effort in post-harvest weight estimation.
